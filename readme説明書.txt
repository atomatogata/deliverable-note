Spring frameworkを利用したWebアプリケーションを作成しています

要件として
ポートフォリオやITのスキルをユーザごとにまとめて記録し、公開できる機能を目標に制作しております。

現時点では、ルーティングと共同開発のために環境づくりをしています
次は、変数の受け渡しと、そのテストを実装予定です

もし、起動する際は、環境変数にuser名とpasswordを登録してください
ユーザ名：DELIVERABLE_USER
パスワード：DELIVERABLE_PASS
環境変数の登録がうまく行かない方はSecurityConfig内の上記の箇所をそれぞれ"user","pass"と書き変えてログインしてください

以下仕様です
言語　Java　11
フレームワーク Spring boot
DB MySQLの予定
フロントについて何の言語を仕様するかはまだ未定です
